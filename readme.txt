# Web Pentest Recon Script

Este script de reconhecimento de penteste web unifica todas as ferramentas e etapas necessárias para uma análise de reconhecimento completa. Ele é adaptável para Windows e Linux, basta instalar as ferramentas necessárias e configurar as variáveis de ambiente corretamente.

## Pré-requisitos

- **Python**: O Linux por default vem com Python pré-instalado. Windows faça e certifique-se de que o Python está instalado e adicionado ao PATH.
- **Nmap**: Ferramenta essencial para varredura de portas.
- **GoBuster**: Utilizado para brute force de diretórios e arquivos web.
- **Wordlist**: Necessária para o uso do GoBuster.

### Instalação

1. **Instalar Python**:
   - No Linux, geralmente já está instalado.
   - No Windows, baixe e instale o Python do [site oficial](https://www.python.org/).

2. **Instalar Nmap**:
   - Linux: `sudo apt-get install nmap`
   - Windows: Baixe e instale do [site oficial](https://nmap.org/).

3. **Instalar GoBuster**:
   - Linux: `sudo apt-get install gobuster`
   - Windows: Baixe e instale do [repositório oficial no GitHub](https://github.com/OJ/gobuster).

4. **Adicionar Ferramentas ao PATH**:
   - Certifique-se de que Python, Nmap e GoBuster estão adicionados ao PATH do sistema.
